# Kosmose korrutuskoloonia PWA

Sisukord:
- index.html
- app.webmanifest
- sw.js
- icons/

## Kuidas kasutada
1. Paki sisu üles veebihostingusse (näiteks GitHub Pages, Netlify või Vercel).
2. Ava saadud link iPhone'is või iPadis Safari brauseris.
3. Vajuta Share -> Add to Home Screen.
4. Pärast esimest avamist töötab mäng ka offline'is.

## Tähtis
- PWA install ei tööta usaldusväärselt lokaalsest .html failist.
- Failid peavad olema veebiaadressi taga (https).
